# schism-client

Relationship management for the people.

## Requirements

-   react
-   redux
-   buble
-   rollup
-   flow

## Development

1.  Fork and clone.
1.  Work on a branch.
1.  Install dependencies with `yarn` (not `npm`).
1.  Start development server.
1.  Hack and make good commits.
1.  Make a pull request.
